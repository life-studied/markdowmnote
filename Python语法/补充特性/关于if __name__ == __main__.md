## 关于`if __name__ == "__main__"`

在python中，如果当前py文件作为程序入口文件，则该文件的隐含变量`__name__`的值为`“__main__”`。否则，为`包名.文件名`。

根据这个特性，可以将文件中的一部分代码放到该判断中，在被其它文件导入时，实现自动隐藏。

**案例**

A模块代码：

```python3
# 模块A

a = 100

print('你好，我是模块A……')

if __name__=='__main__':		#被其它模块调用时，不运行

    print(a)
```

B模块代码：

```python3
# 模块B

from package01 import A

b = 200

print('你好，我是模块B……')

print(b)
```

执行B模块，输出结果如下：

```python3
你好，我是模块A……
你好，我是模块B……
200
```
