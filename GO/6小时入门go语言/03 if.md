# 03 if

​	下面是if语法的简单用法。

```go
// Package main -----------------------------
// @file      : if.go
// @author    : Yunyin
// @contact   : yunyin_jayyi@qq.com
// @time      : 2023/7/27 10:41
// -------------------------------------------
package main

import "fmt"

func main() {

	if a, b := 10, true; a > 5 {
		fmt.Println("a > 5")
	} else if b {
		fmt.Println("b is true")
	} else {
		fmt.Println("final")
	}

}

```
