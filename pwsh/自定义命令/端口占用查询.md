# 端口占用查询

## 批量查找空闲端口

```powershell
[void]($busy = [System.Collections.Generic.HashSet[int]]::new([int[]]((Get-NetTCPConnection).LocalPort + (Get-NetUDPEndpoint).LocalPort))); (1024..65535 | ? { !$busy.Contains($_) } | Select -First 100) -join ', '
```

* 1024..65535：查询范围
* 100：显示前100个
