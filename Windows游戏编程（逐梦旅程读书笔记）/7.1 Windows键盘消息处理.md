## Windows键盘消息处理

> 在Windows下，所有按键都会被映射为一个虚拟键，有自己的虚拟键码。
>
> message存储了消息类型，而实际的消息在另外的参数里。

#### 1.键盘消息处理

在消息处理函数中，键盘消息处理的情况：参数`wParam`存储了传入的实际按键。下面是一个例子。

```C++
switch(message)
{
    case WM_KEYDOWN:
        switch(wParam)
        {
            case VK_UP:
                ...;
                break;
            case VK_ESCAPE:
                ...;
                break;
            default:
                break;
        }
        
    case WM_DESTROY:
       	...;
        break;
        
    default:
        break;
}
```
