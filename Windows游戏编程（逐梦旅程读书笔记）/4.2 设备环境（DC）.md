## 设备环境（DC）

#### 使用方法一（BEGINPAINT和ENDPAINT）

> 在窗口过程函数里，WM_PAINT的case后使用。

流程：

* 创建一个PAINTSTRUCT
* 调用BeginPaint函数
* 将绘画信息装入PAINTSTRUCT
* 调用EndPaint函数，完成绘画

```C++
HDC g_hdc;
...
case WM_PAINT:
	g_hdc = BeginPaint(hwnd, &paintStruct);
	...
    EndPaint(hwnd, &paintStruct);
	break;
```

#### 使用方法二（GETDC和RELEASEDC）

```C++
HDC g_hdc;
g_hdc = GetDC(hwnd);
/*
此处使用g_hdc为媒介进行
*/
ReleaseDC(hwnd, g_hdc);
```
