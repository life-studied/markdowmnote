## 控制语句

#### 循环(continue,break可用)

##### for

> for i = 1:10
>
> ​	v(i)=2^i;
>
> end;

##### while

> i=1;
>
> while i<=5
>
> ​	v(i)=100;
>
> ​	i=i+1;
>
> end;

#### 控制

##### if

> if i==5
>
> ​	disp('the value is 5');
>
> elseif i==6
>
> ​	disp('the value is 6');
>
> else
>
> ​	disp('the value is 0');
>
> end;





#### 函数

格式：

```matlab
function 返回值=函数名（传入参数）
函数体;
```

**例如**

```matlab
function y=squarethe_num(x)
y=x^2;
```

```matlab
function [x,y]=test(x)
y=x;
x=1;
```
