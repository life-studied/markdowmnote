# 案例

## 匹配RGB值

```txt
#00
#ffffff
#ffaaff
#00hh00
#aabbcc
#000000
#ffffffff
```

​	对于上述RGB值，匹配为：

```regex
#[a-fA-F0-9]{6}\b
```

​	注意：`\b`表示单词的边界

## IP地址匹配

### 提取IP地址

​	如果只是提取，不保证IP地址的合法性，可以使用：`\d\.\d\.\d\.\d`来提取

### 匹配IP地址

​	如果要匹配合法的IP地址，则是：

```regex
\b((25[0-5]|2[0-4]\d|[01]?\d\d?)\.){3}(25[0-5]|2[0-4]\d|[01]?\d\d?)\b
```
