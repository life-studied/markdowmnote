# 03.3 UE类的命名规则

​	按照虚幻引擎的命名规则，添加的命名前缀。常用的前缀如下： 

| 前缀 | 说明                           |
| ---- | ------------------------------ |
| F    | 纯C++类                        |
| U    | 继承自UObject，但不继承自Actor |
| A    | 继承自Actor                    |
| S    | Slate控件相关类                |
| H    | HitResult相关类                |

​	虚幻引擎头文件工具`Unreal Header Tool`（UHT）会在编译前检查你的类命名。

​	如果类的命名出现错误，那么它会提出警告并终止编译。
