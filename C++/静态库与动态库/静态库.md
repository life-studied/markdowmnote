### 使用静态库

#### 1. 设置项目属性

* 添加库目录
  * 将生成的静态库路径添加到库目录
* 链接库文件
  * 在项目属性中设置文件名
  * 或者使用`#pragma comment(lib, “libname.lib”)`
* 编写函数声明
  * `extern` + 函数声明
  * 或者使用头文件

### 案例1（`extern` + 函数声明）

```C++
//staticLib.cpp
int add(int a, int b)
{
    return a + b;
}

//main.cpp
#pragma comment(lib, "staticLib.lib")
extern int add(int,int);

int main()
{
    add(1, 2);
}
```

### 案例2（或者使用头文件）

```C++
//staticLib.h
int add(int, int);

//staticLib.cpp
#include "staticLib.h"
int add(int a, int b)
{
    return a + b;
}

//main.cpp
#include "staticLib.h"
#pragma comment(lib, "staticLib.lib")

int main()
{
    add(1, 2);
}
```
