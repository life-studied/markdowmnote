## 8.3 默认参数

介绍：函数省略实参时默认使用的值

##### 设置方法及要求

设置方法：仅在**函数原型**的参数列表中赋值

```c++
char * left(const char * str, int n=1);

char * left(const char * str, int n)
{
    ...
}
```

要求：默认参数必须从右往左添至参数列表

```c++
int harpo(int n,int m=4,int j=5);
```

##### 使用说明

对于带n个默认参数的共m个参数的函数，使用时参数列表中可有[m-n,m]个参数，根据情况使用
