## 7.5 在STL中使用lambda表达式

​	在stl的算法函数中，常常需要传入一个谓词来作为回调函数使用，此处就可以使用lambda表达式来代替辅助函数。

```C++
#include <iostream>
#include <vector>
#include <algorithm>

int main()
{
    std::vector<int> x = {1,2,3,4,5};
    std::cout
        <<*std::find_if(x.cbegin(),x.cend(),[](int i){return (i%3)==0;})
        <<std::endl;
}
```
