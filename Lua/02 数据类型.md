# 02 数据类型

| 数据类型 | 说明             | 备注                                                         |
| -------- | ---------------- | ------------------------------------------------------------ |
| nil      | 无效值，false    |                                                              |
| boolean  | true false       |                                                              |
| number   | 双精度浮点数     |                                                              |
| string   | 字符串           | 单引号<br />双引号<br />多行字符串使用[[]]                   |
| table    | 数组/映射        |                                                              |
| function | C或Lua编写的函数 |                                                              |
| thread   | 协同线程         |                                                              |
| userdata | 用户自定义类型   | 表示一种用C/C++创建的类型<br />可以将任意C/C++类型存储到Lua变量中 |
