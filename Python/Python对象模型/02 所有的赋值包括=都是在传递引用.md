# 所有的赋值包括=都是在传递引用

在了解Python的对象模型的基础上，就可以得出这个结论：所有的赋值包括=都是在传递引用。

这里的赋值包括：

* =符号
* 函数传参
* 函数返回值
* list的copy()
* list切片
* 其它赋值操作

所有赋值的操作（a=b），语义都是将等号右边变量的标签，传给等号左边的变量。

## list的copy与切片

对于list来说，实际就是将每一个元素进行了a=b的操作。其它没什么好说的。

## 函数传参和返回值

人们总是会对Python函数传参非常迷惑，并且网上的说法也很奇怪，扯了很多概念。

但是实际上**就是在传递引用**，或者说是那个标签。

一个简单的代码，你可以用任意的类型去测试：

```python
a = 1	# any type
print(id(a))

def test(data):
    print(id(data))
    return data

print(id(test(a)))
```
