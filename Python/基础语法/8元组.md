## 元组

> 与列表类似，但是不支持修改元素

#### 创建

```python
rhyme = (1,2,3,4,5,"hello")		#使用（）
rhyme = 1,2,3,4,5,"hello"		#不带（）
```

#### 自带方法如下

```python
rhyme.count(3)		#查询元素个数
rhyme.index(3)		#获取下标
...
```

#### 一些细节

##### 创建一个变量的元组

```python
x=(520)		#int
x=(520,)	#tuple
```

##### 元组可变的注意点

> 虽然元组不可变，但是如果内部是一个数组，那么元组存储的本质是地址，至于地址内的值与之无关，所以在元组中的列表是可变的

#### 打包和解包

```python
x=("123",1,2,3,3.5)			#将数据组合
a,b,c,d,e=x					#将x中的每个元素赋给对应的变量
#注：只要是序列都能实现解包		例如：列表，字符串
#注:左右数目必须一致
```
