# podman镜像代理配置

podman的配置和docker不太一样，路径的前缀不需要https。

```shell
sudo nano /etc/containers/registries.conf
```

添加代理（根据时间去替换）：

```toml
unqualified-search-registries = ["docker.io"]

[[registry]]
prefix  = "docker.io"
location = "docker.io"

[[registry.mirror]]
location = "docker.1panel.live"
[[registry.mirror]]
location = "docker.1ms.run"
```
