## UPDATE

#### 1.WHERE子句

用于筛选符合条件的记录

```mysql
WHERE score > 20 and score < 40;
```

#### 2.UPDATE

##### 语法

```mysql
UPDATE 表名 SET 列名1=值1, 列名2=值2, 列名3=值3, ... WHERE 条件; 
```

##### 分析：

* ```mysql
	UPDATE 表名 SET 列名1=值1, 列名2=值2, 列名3=值3, ...	-- 将对应的列值改为设置的值
	```

* ```mysql
	WHERE 条件; 	-- 筛选条件
	```

##### 例子

将course表中的数据库的学分改为4，学时改为15

```mysql
UPDATE course SET score=4, `time`=15 WHERE name="数据库";
```
