## 创建计算字段

#### 1.字段

> 检索出的值以一种格式化形式输出

##### 1.1拼接字段

> 使用concat函数，参数为拼接的部分字段

```mysql
SELECT CONCAT(stu_name,stu_id) as name_id, stu_status
From student
order by stu_id;
```

> 使用as别名来创建对应的结果列名

##### 1.2算术计算

> 使用算术运算符

```mysql
SELECT stu_id, gra_score*2 as grade
From student
order by stu_id;
```
