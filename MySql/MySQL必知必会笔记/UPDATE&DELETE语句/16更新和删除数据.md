## 更新和删除数据

#### 1.更新数据

> 使用UPDATE语句更新表中数据。

##### 1.1更新单列

>语句由三部分组成
>
>1.要更新的表名
>
>2.列名与它的新值
>
>3.更新列的过滤条件

```mysql
UPDATE Customers
SET cust_email = 'my@s.com'
Where cust_id = '00000001';
```

**注：如果没有where过滤，则对象为所有行**

##### 1.2更新多列

> SET语句中使用逗号分隔条件

```mysql
UPDATE Customers
SET cust_email = 'my@s.com',
	cust_name = 'Jay'
Where cust_id = '00000001';
```

##### 1.3删除某个列

> 将值设置成NULL即可。

#### 2.删除数据

> 使用DELETE FROM语句删除

```mysql
DELETE FROM Customers
Where cust_id = '00000001';
```
