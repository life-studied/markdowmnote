# 基本操作命令

## 控制台

​	控制台模拟了一个类似Linux终端的内容，但是实际有所差别。

```miniscript
/
|__net
|__sys
	|__demo
|__usr
```

### 切换目录

```miniscript
cd "dir name"
```

### 查看内容

```miniscript
dir
```

### 加载脚本

​	脚本是`xxx.ms`。加载只需前缀即可。

```miniscript
load "scriptname"
```

### 修改

​	加载的前提下。

```miniscript
edit
```

### 保存

​	加载的前提下。

```miniscript
save
```

### 执行

​	加载的前提下。

```miniscript
run
```
